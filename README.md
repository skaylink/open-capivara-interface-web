# open-capivara-interface-web
For the main repo, see [open-capivara](https://github.com/skaylink/open-capivara)

Project setup:
```
npm install
```

 Compiles and hot-reloads for development:
```
npm run serve
```

 Compiles and minifies for production:
```
npm run build
```

 Run your tests:
```
npm run test
```

 Lints and fixes files:
```
npm run lint
```

 Run your unit tests:
```
npm run test:unit
```
